# Well-Known Port Reference

Interactive reference for the top 100 well-known TCP/UDP port numbers. Search by port number, service name, or description. Everything runs client-side.

**[Live Demo](https://gmowses.github.io/port-reference)**

## Included Ports

20, 21, 22, 23, 25, 53, 67, 68, 80, 110, 119, 123, 135, 137, 139, 143, 161, 162, 179, 389, 443, 445, 465, 514, 515, 587, 636, 993, 995, 1080, 1433, 1434, 1521, 1723, 2049, 2082, 2083, 2086, 2087, 3306, 3389, 4443, 5060, 5061, 5432, 5900, 5938, 6379, 6443, 8080, 8443, 8888, 9090, 9200, 9300, 11211, 27017, 27018

## Features

- Search by port number, service name, or description
- Filter by protocol: TCP, UDP, TCP+UDP
- Filter by category: Web, Email, Database, Remote, Security, DNS/DHCP, Monitoring, Files, Messaging
- Expandable rows with RFC references
- Encryption indicator (encrypted, TLS/STARTTLS, cleartext)
- Dark/light mode
- i18n: English and Portuguese (BR)
- Client-side only

## Tech Stack

React 19 + TypeScript, Tailwind CSS v4, Vite, Lucide icons, GitHub Pages
